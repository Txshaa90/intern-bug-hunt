# E-Commerce Intern Debug Tasks

This folder contains 5 separate front-end debugging exercises for web developer interns.

Each task includes:
- `index.html`
- `styles.css`
- `script.js`

Instructions for interns:
1. Open each folder and run the page in a browser.
2. Inspect the HTML, CSS, and JavaScript.
3. Find the bugs and fix them.
4. Document the issues you found and how you fixed them.

Suggested evaluation areas:
- Debugging accuracy
- Code reading skills
- HTML/CSS/JavaScript fundamentals
- Attention to detail
- Explanation of fixes

Task list:
- `task-01-product-page`
- `task-02-cart-page`
- `task-03-checkout-form`
- `task-04-product-filter`
- `task-05-promo-summary`
